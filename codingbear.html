Games

Dungen is a work in progress to create a turn-based dungeon crawling RPG with procedurally generated maps, written in Javascript. It's only partially tested on Linux and Android with Firefox and Chrome, versions:
- Debian Stretch
- Android 5.0.2
- ChromeOS/Chrome/Chromium 49
- Firefox 45
The only issue I've found so far is that I've used vogg orbis for the audio and Chrome on Android doesn't seem to support it yet. The ECMA 6 language features appear to work nicely though.
